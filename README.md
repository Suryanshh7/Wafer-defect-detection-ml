# Wafer Defect Detection using Machine Learning

## 📌 Project Overview
This project focuses on detecting semiconductor wafer defects using machine learning techniques based on process parameters such as temperature, pressure, gas flow, and etch rate.

---

## 🎯 Objective
To predict whether a wafer is defective using fabrication process data.

---

## 📊 Dataset Insights
- Total samples: 5000
- Defective wafers: 7 (~0.14%)
- Highly imbalanced dataset

---

## ⚠️ Challenges
- Extreme class imbalance
- Very few defect samples
- No clear separation in features

---

## 🛠️ Methods Used
- Random Forest Classifier
- SMOTE (for imbalance handling)
- Isolation Forest (Anomaly Detection)

---

## 📈 Results
- High accuracy due to imbalance
- Poor defect detection performance
- Highlighted limitations of dataset

---

## 🧠 Key Learnings
- Importance of data quality over model complexity
- Challenges in real-world semiconductor datasets
- Need for anomaly detection techniques

---

## 🚀 Future Work
- Use wafer map images (CNN-based approach)
- Collect more defect samples
- Apply advanced anomaly detection methods
